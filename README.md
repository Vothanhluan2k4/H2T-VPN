## Local Setup & Development

Follow these steps to run the application locally:

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Local Development Server
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the interactive page.

### 3. Build for Production
To verify compilation and bundle size:
```bash
npm run build
```
The output bundles will be optimized and saved to the `dist/` directory.

## Core Visual & Interaction Features
* **Floating Island Navigation:** Detached glass menu bar with staggered overlay navigation on mobile.
* **Bento Grid Rankings:** Unbiased comparative layout ranking the top 3 global VPNs.
* **Interactive Speed Simulator:** Simulate real-time cryptographic handshakes, ping, and WireGuard protocol speeds using dynamic network nodes and SVG charts.
* **Responsive Layouts:** Designed mobile-first, ensuring all grids collapse cleanly below `768px` for optimal touchscreen navigation.
* **No Em-Dash Constraint:** Adhered strictly to professional design copywriting guidelines by banning stylistic em-dashes.
